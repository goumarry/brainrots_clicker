📋 Cahier des Charges — BrainrotClicker
Un Idle/Clicker Game inspiré de Clicker Heroes, univers Brainrot
🧠 1. CONCEPT & VISION
1.1 Pitch

    Un clicker/idle game où le joueur doit vaincre des ennemis issus de la culture brainrot (Skibidi Toilet, Sigma, Ohio, Fanum Tax, Rizz, etc.) en cliquant et en recrutant des héros brainrot qui infligent des dégâts automatiques (DPS). Progression infinie, prestige, et montée en puissance addictive.

1.2 Cible

    Joueurs casual 13-25 ans
    Fans de memes / culture brainrot
    Plateforme : Web (CrazyGames)

1.3 Références

    Clicker Heroes
    Cookie Clicker
    Tap Titans

2.2 Mécaniques Principales
🖱️ A) Le Click
Élément	Détail
Click Damage (CPC)	Dégâts de base par click, augmentables
Critical Click	% de chance de crit (x3 à x10)
Click Storm	Skill temporaire : auto-click x20/sec pendant 30s
Combo Counter	Multiplicateur qui monte avec les clicks rapides
🦸 B) Les Héros Brainrot (système central)

Chaque héros a :

    Coût d'achat (scaling exponentiel)
    DPS de base
    Niveau (upgradable à l'infini)
    Milestones (tous les 25 niveaux → unlock skill passif ou boost)
    Rareté : Common / Rare / Epic / Legendary / Mythic

Roster initial (15-20 héros) :

👾 C) Les Ennemis

    Ennemis normaux : 10 par zone, HP scaling
    Boss : tous les 5 niveaux, timer de 30s pour le tuer
    Boss Raid (tous les 50 niveaux) : boss spécial avec mécanique unique

Formule HP des ennemis :

text

HP(zone) = base_hp * 1.55^zone * (1 + floor(zone/5) * boss_mult)

Exemples d'ennemis :

    Zone 1-10 : NPC Normies
    Zone 11-25 : Cursed TikTokers
    Zone 26-50 : Skibidi Army
    Zone 51-100 : Ohio Monsters
    Zone 100+ : Sigma Overlords
    Zone 200+ : Brainrot Titans
    Zone 500+ : Ascended Memes

⚡ D) Skills Actifs (Cooldown-based)
Skill	Effet	Cooldown	Durée
Skibidi Rush	Auto-click x30	3 min	30s
Sigma Grindset	DPS x5	5 min	60s
Ohio Mode	Tout peut arriver (random buff x2-x100)	10 min	15s
Fanum Tax	Vole 10% HP de l'ennemi	2 min	Instant
Rizz Aura	Gold x10	8 min	45s
GYATT Bomb	Nuke (damage = 10min de DPS)	15 min	Instant
Mewing Focus	Crits garantis	5 min	20s
🔄 E) Système de Prestige — "BRAINROT ASCENSION"
Élément	Détail
Condition	Disponible à partir de la zone 100
Monnaie	Sigma Souls (earned based on zone reached)
Reset	Gold, niveaux héros, zone — tout reset
Keep	Sigma Souls, Ascension upgrades, achievements
Arbre de talents	5 branches avec upgrades permanents

Formule Sigma Souls :

text

souls = floor((zone_max - 75) * 0.5 * (1 + prestige_bonus))

Arbre d'Ascension (5 branches) :

text

           🧠 BRAINROT TREE
          /    |    \    \     \
     💪      💰     ⚡    🎯     🌀
    SIGMA   GRIND  SPEED  CRIT  CHAOS
    
💪 Sigma Path    → +DPS%, +Hero damage
💰 Grind Path    → +Gold%, +Idle earnings  
⚡ Speed Path    → -Cooldowns, +Click speed
🎯 Crit Path     → +Crit%, +Crit damage
🌀 Chaos Path    → Random procs, Lucky drops

🎁 F) Systèmes Secondaires

1. Achievements (100+)

    "First Blood" → Kill 1 enemy
    "Skibidi Slayer" → Kill 1000 Skibidi
    "Sigma Grindset" → Reach zone 100 without buying heroes
    Chaque achievement donne un bonus permanent

2. Reliques (Drop des boss)

    Items passifs avec stats random
    Rareté : Bronze / Silver / Gold / Diamond
    Exemple : "Skibidi Plunger" → +15% DPS, +5% Gold

3. Daily Rewards / Quests

    Login streak → rewards croissants
    3 daily quests : "Kill 500 enemies", "Reach zone X", etc.
    Reward : Gems (premium currency F2P friendly)

4. Offline Progress

    Le jeu calcule les gains pendant l'absence
    Max 8h d'idle earnings (upgradable via Ascension)

5. Mercenaires (Late game)

    Envoyer des héros en mission pour du loot
    Missions de 1h à 24h

2.3 Économie & Balancing

text

Gold earning rate:
gold_per_kill = base_gold * zone^1.5 * gold_multiplier

Hero cost scaling:  
cost(level) = base_cost * 1.07^level

DPS scaling:
hero_dps(level) = base_dps * level * (1 + milestone_bonus)

Zone HP scaling:
enemy_hp(zone) = 10 * 1.55^zone
boss_hp(zone) = enemy_hp(zone) * 10

4.3 Justification détaillée
Choix	Raison
Phaser 3	Framework mature, WebGL+Canvas, parfait pour 2D, énorme communauté, compatible CrazyGames nativement, gestion sprites/animations/input/scenes built-in
TypeScript	Type safety indispensable pour un jeu avec beaucoup de data (héros, upgrades, formules), autocomplétion, refactoring facile
Vite	Build ultra rapide, HMR, output optimisé, tree-shaking, facile à config
break_infinity.js	Librairie pour gérer les nombres au-delà de 1e308 (indispensable pour un idle game), supporte les opérations mathématiques
Howler.js	Audio cross-browser fiable, sprite audio, pool, WebAudio API
LocalStorage	Save rapide côté client, suffisant pour MVP
CrazyGames SDK	Obligatoire pour ads, events, leaderboard, game loading

4.5 Patterns Architecturaux

typescript

// 🔥 Event-Driven Architecture
// Tous les managers communiquent via un EventBus

// EventBus.ts
type GameEvent = 
  | 'enemy:killed' 
  | 'enemy:spawned'
  | 'hero:purchased' 
  | 'hero:upgraded'
  | 'hero:milestone'
  | 'zone:changed' 
  | 'zone:boss'
  | 'gold:changed' 
  | 'gold:earned'
  | 'skill:activated' 
  | 'skill:ready'
  | 'click:damage'
  | 'click:critical'
  | 'ascension:start'
  | 'ascension:complete'
  | 'achievement:unlocked'
  | 'relic:dropped'
  | 'save:complete'
  | 'offline:calculated';

// 🔥 State Machine pour les ennemis
enum EnemyState {
  ENTERING,    // Animation d'entrée
  IDLE,        // Attend les dégâts
  HIT,         // Flash quand touché
  DYING,       // Animation de mort
  DEAD         // Transition vers suivant
}

// 🔥 GameState centralisé (Single Source of Truth)
interface GameState {
  gold: Decimal;
  totalGold: Decimal;
  currentZone: number;
  maxZone: number;
  enemyIndex: number;
  heroes: HeroState[];
  skills: SkillState[];
  ascension: AscensionState;
  relics: RelicState[];
  achievements: string[];
  stats: PlayerStats;
  settings: GameSettings;
  lastSaveTime: number;
}
